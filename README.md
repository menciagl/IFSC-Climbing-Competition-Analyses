# Analysis of IFSC World Cup Winners over the Last Decade (2014-2024)

The IFSC Climbing World Cup is an annual series of competitive climbing events held across different locations worldwide, organized by the **International Federation of Sport Climbing (IFSC)**. This report provides an analysis of the **overall annual winners of the IFSC World Cup from 2014 to 2024**, based on the total points accumulated throughout each season.

The repository contains a **file with the data in .xlsx format** taken from the IFSC website and **a file in .R format with the descriptive analyses** performed.

Examining public data of the IFSC World Cup champions and their characteristics is crucial for athletes, coaches, and sports analysts, as it allows stakeholders to identify factors contributing to success and **predict future competitive dynamics**.
